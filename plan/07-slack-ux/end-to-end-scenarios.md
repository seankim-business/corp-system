## 시나리오 2: 컬렉션 론칭 플랜 (멀티 에이전트)

### 상황
Product Lead가 신규 컬렉션 론칭 전체 플랜을 요청합니다.

### 대화 흐름

```
┌─────────────────────────────────────────────────────────────────────────┐
│ #obj-2025-ss-launch                                                     │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│ 👤 @이프로덕트                                                  09:00 AM │
│ @company-os 이번 S/S 컬렉션 론칭 플랜 세워줘.                           │
│ 론칭일 3월 1일 기준으로 전체 일정이랑 각 팀 담당 업무 정리해줘.          │
│                                                                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│ 🤖 Company OS                                                   09:00 AM │
│ :robot_face: *[System]*                                                 │
│ 🚀 *S/S 컬렉션 론칭 플랜 수립 시작*                                      │
│                                                                         │
│ 이 작업은 여러 팀의 협업이 필요합니다.                                   │
│                                                                         │
│ **참여 에이전트**                                                       │
│ • :package: Product Agent (총괄)                                        │
│ • :art: Brand Agent (콘텐츠)                                            │
│ • :gear: Ops Agent (물류)                                               │
│ • :chart: Finance Agent (예산)                                          │
│                                                                         │
│ _진행 상황은 스레드에서 확인하세요_ 🧵                                   │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```
